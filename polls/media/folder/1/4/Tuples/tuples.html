Tuples

Tuple is similar to a list.The only difference between both is that,elements of a tuple cannot be changed once it is assigned whereas in a list, elements can be changed. Tuples use paranthesis(), whereas lists use square brackets[].

Advantages of Tuples over Lists

The following are few advantages of implementing tuple over a list:

  • Tuple enhances performance, since iterating through tuple is faster than with list.
  • Tuples are used for different datatypes and lists are used for similar datatypes.
  • Implementing a data that doesn't change as a tuple will guarantee that it remains write-protected.

Creating a Tuple

  • A tuple is created by putting different comma-separated values in between parantheses which is optional.
  • Tuple elements can be of any type,for example,integer,float,string etc and can have any number of items.

Examples:

        my_tuple = ("fruit", 1, 6.7)

        print(my_tuple)

Output:

        ('fruit', 1, 6.7)

If we give an empty tuple,it gives empty parantheses as output

        my_tuple = ( )

        print(my_tuple)

Output:

        ( )

Inorder to write a single value in a tuple,we need to include comma after the value

        my_tuple = ("fruit",)

        print(my_tuple)

Output:

        ('fruit',)

Nested Tuple

It can contain any type. For example: int, float, string, list, tuple

        my_tuple = ("fruit", [1,2,3], (4,5))

        print(my_tuple)

Output:

        ('fruit', [1,2,3], (4,5))

Inorder to know the type of the variable just give 'type' with the variable

        my_tuple = ("fruit",)

        print(type(my_tuple))

Output:

        < class 'str' >

Accessing Values in Tuples

To access values in a tuple, indexing can be used.Inorder to access a value in the tuple,use square brackets along with the index of the value..

Indexing

Indexing starts from Zero(0).So, a tuple having 4 elements will have index from 0-3

        my_tuple = (4,5,6,7)

        print(my_tuple[1])

Output:

        5

Negative Indexing

Negative Indexing starts from Right using negative integers(-1,-2,-3...).So, a tuple having 4 elements will have index -1 to -4

        my_tuple = (4,5,6,7)

        print(my_tuple[-1])

Output:

        7

Index Error        

Accessing the elements that are out of the index range will raise an Index out of range error

        my_tuple = (4,5,6,7)

        print(my_tuple[6])

Output:

        Index Error: Index out of range

Type Error

Index must be an integer,using float or other types will raise Type error

        my_tuple = (4,5,6,7)

        print(my_tuple[1.0])

Output:

        TypeError: list indices must be integers, not float

Accessing values in Nested Tuple

Inorder to get the value of a tuple from the nested tuple,first give the index of the tuple value and then give the index of the value that is to be retrieved

        my_tuple = ("fruit", [1,2,3], (4,5))

        print(my_tuple[1][1])

Output:

        2

Slicing

A range of items in a tuple can be accessed using a slicing operator ":"

[1:4] ---> including 1 and excluding 4

        my_tuple = ('p', 'q', 'r', 's')

        print(my_tuple[1:])

Output:

        ('q', 'r', 's')

Updating Tuples

Tuples are immutable which means one cannot update or change the values of tuple elements

        my_tuple = ("fruit","ate","sweet")

        my_tuple[1]="bottle"

Output:

        tuple object does not perform item assignment

Changing Tuples

Tuples elements cannot be changed but if the element is mutable datatype like lists,then its items can be changed

        my_tuple = ("richie",4,5,[1,2,3])

        my_tuple[1]=9

        print(my_tuple)

Output:

        ("richie",4,5,[9,2,3])

Deleting Tuples

Deleting individual elements from a tuple is not possible but can delete entire tuple which is done through "del" statement

        my_tuple = ("fruit","ate","sweet") ------> tuple item deletion

        del my_tuple[1]

Output:

tuple object does not support item deletion

        my_tuple = ("fruit","ate","sweet") ------> tuple deletion

        del my_tuple

Output:

        my_tuple is deleted

Tuple Operations

The following are some of the tuple operations:

Length

Gives the length of the tuple

        my_tuple = ("fruit","ate","chocos")

        print(len(my_tuple))

Output:

        3

Concatenation

combines the two tuples

        tup1 = ("fruit","ate","chocos")

        tup2 = ("monkey","ate","banana")

        print((tup1) + (tup2))

Output:

        ("fruit","ate","chocos","monkey","ate","banana")

Repeat

Repeats the values of the tuples

        my_tuple = (("You fruit",) * 3)

        print(my_tuple)

Output:

        ('You fruit','You fruit','You fruit')

Tuple Functions

The following are some of the tuple functions:

Max

returns the maximum valued element

        my_tuple = (1, 2, "ant", "fruit")

        print(max(my_tuple))

Output:

        4

Min

returns the minimum valued element

        my_tuple = (1, 23, "ant", "fruit")

        print(min(my_tuple))

Output:

        1

Sorted

returns a sorted list of the tuple value

        my_tuple = ("bottle")

        print(sorted(my_tuple))

Output:

        ['b', 'e', 'l', 'o', 't', 't']
