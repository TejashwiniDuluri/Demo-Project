LISTS

Definition

A list is a collection of identical or different type of elements which are enclosed in square braces '[ ]' with comma saperated sequence.

Something More...

  • In List index starts with zero from first element.
  • List are Mutable. which means they can be updated or modified once defined.

Syntax:

MyList=[] -------> Empty List

MyList1=['x', 'y'] -------> List of particular datatypes

MyList2=['x', 'y', 5] --------> List of different datatypes

One in Another (Nested List)

['a','b'] ['x', 'y', 5] --------> ['x', 'y', 5, ['a','b']]

Updating List

We can change list elements using indexing and =(assignment operator).

myList = [1,2,3,4]

myList[0]='a'

myList

Output: ['a',2,3,4]

Note: When a list is assigned to another variable it make both of them pointing to the same memory I.e. doesn't create separate memory.

Slicing in List

Range of elements in a list can be accessed using sclicing.

Syntax:

listName[starting Index:ending Index]

Example

myList = [1,2,3,4,5,6,7,8]

myList[2:4] ------> indicate elements starting from 2nd index element to 3rd index leaving 4th and after elements.

myList[0:] ------> indicate all elements starts from beginning.

myList[:] ------> all elements.

myList[:4] ------> from beginning to 3rd index element.

Deleting Lists and its Elements

To delete a list 'del' keyword is used. To remove an element from a list use 'del' keyword along with list indexing or 'remove()' along with element.

Deleting an element

iba=['i', 'b', 'a']

del iba[0] ------> deletes element of list at 0th index

Output: ['b', 'a']

iba.remove('b') ------> deletes element of list if match is found

Output: ['a']

Deleting a List

iba=['i', 'b', 'a']

Operations and Methods

.pop()

pop() method is used to remove an element at given index else removes last element.

iba=['i', 'b', 'a']

iba.pop() ------> removes last element from the list

Output: ['i', 'b']

iba.pop('i') ------> removes specific element from the list

Output: ['b']

len()

Helps to find length of list.

len(iba)

Output: 3

.append()

This method takes any value as parameter and appends to the list.

iba.append('a')

Output: ['b','a']

.clear()

This method removes all elements from list.

iba.clear()

BULT-IN-FUNCTIONS

sum()

Returns sum of all elements in list (applicable only to list of numbers).

iba=[1,5,6]

sum(iba)

Output: 12

min()

Finds the minimum value of a list.

min(iba)

Output: 1

max()

Finds the maximum value of a list

max(iba)

Output: 6

sorted()

Returns a newly sorted list.
