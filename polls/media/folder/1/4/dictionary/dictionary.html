Dictionary

It is similar to hash (or) maps in other languages. It consists of key value pairs. The value can be accessed by unique key. It is a collection which unordered, changeable and indexed written in curly brackets {} .

Create a dictionary

It is simple as placing items inside curly brackets {} separated by comma (,).

Syntax :

        dict = { key : value }

Value can be any data type & can repeat, keys must be immutable type and must be unique.

Example:

        my_dict = {} ------------> Empty

        my_dict = { 1 : 'apple' } ------------> Integer keys

        my_dict = { 'name' : 'john' , 1 : [ 1, 2, 3] ------------> Mixed keys

        my_dict = dict({ 1 : ' apple '}) ------------> using dictionary

        my_dict = dict (( 1 , 'apple' )] ------------> Sequence having each item as pair

Access elements from a dictionary :

While indexing is used with other container types to access values, dictionary uses keys, Keys can be used either inside [] brackets OR by get() method.

Example:

        my_dict = { 'name' : 'Raji', 'age' : 21 }

        print(my_dict['name'])

Output : 

        Raji

        print(my_dict['age'])

Output :

         21

Change or Add element in Dictionary :

Dictionary are mutable using assignment operator we can add items OR change value of items. If keys is already present then values get updated, else a new key: value pair is added.

Example:

        my_dict = { 'name' : 'Raji', 'age' : 21 }

        my_dict = { 'name' : 'Teju' }

        print(my_dict)

Output :

        { 'name' : 'Teju' , 'age' : 21 }

        my_dict = { 'address' : 'hyd' }

        print ( my_dict )

Output :

        { ' name' : 'Teju' , 'age' : 21 , 'address' : 'hyd' }

Appending elements to a Dictionary :

If suppose you want to add extra element to your already initialized list which has elements, then

Example :

        my_dict = { 'name' : 'Raji', 'age' : 21 }

        my_dict[ 'email ' : 'abc@gmail.com' ]

        print(my_dict)

Output :

        { 'name' : 'Raji', 'age' : 21, 'email ' : 'abc@gmail.com'}

Delete or Remove Elements from a Dictionary :

By using pop() , we can remove a particular item in dictionary. This method removes as item with key & returns values.

Example:

        my_dict = { 'name' : 'Raji',1:4, 'age' : 21 }

        print(my_dict.pop(1) )

Output :

        4

        print(my_dict)

Output :

        { 'name' : 'Raji' , 'age' : 21 }

POP () Items :

we can remove and return an arbitrary item ( key, value ) from the dictionary

        my_dict = { 1 : 1 , 2 : 2 , 3 : 3 }

        print(my_dict.pop(1))

Output :

        (1, 1)

        print(my_dict)

Output :

        { 2 : 2 , 3 : 3 }

Clear () :

Is used to remove all item at a time.

        my_dict = { 1 : 1 , 2 : 2 , 3 : 3 }

        my_dict.clear()

        print(my_dict)

Output :

        { }

Delete () :

keyword to remove individual item OR entire dictionary itself.

        my_dict = { 1 : 1 , 2 : 2 , 3 : 3 }

        del my_dict[1]

        print(my_dict)

Output :

        {2 : 2 , 3 : 3 }

Properties :

Dictionary values have no restrictions. They can be either Standard OR uses defined objects.

Key Points :

  • More than 1 entry per key not allowed means no duplicated key. If duplicated key en-counted during assignment, the last assignment wins.

Example:

my_dict = { 'name' : 'Raji', 'age' : 21 }

print ( my_dict ['name'])

Output :

        Raji

  • Keys must be immutable, means we can use strings, number OR tuple as dictionary keys but [ ' key ' ] not allowed.

Example:

my_dict = { 'name' : 'Raji', 'age' : 21 }

print ( my_dict ['name'])

Output :

        Type Error: unhashable type: 'list'

Nested Dictionary :

A nested dictionary is a dictionary inside a dictionary. It is a collection of dictionary into one single dictionary.

Syntax:

        nested_dict = { 'dict1', 'dict2' }

Create a Nested Dictionary :

        my_dict = { 1 : {'name' : 'Raji', 'age' : 21 }, 2 : { 'name' : 'teju'}}

        print(my_dict)

Output :

        { 1 : {'name' : 'Raji', 'age' : 21 }, 2 : { 'name' : 'teju'}}

Access the element of a Nested Dictionary :

        my_dict = { 1 : { 'name' : 'Raji', 'age' : 21 }, 2 : { 'name' : 'nandu' }}

        print(my_dict[2]['name'])

Output :

        nandu

Add element to a Nested Dictionary :

        my_dict = { 1: { 'name' : 'Raji', 'age' : 21 }, 2 : { 'name' : 'nandu' }}

        dict [3] = { }

        my_dict[3]['name']='Gopi'

        print(my_dict)

Output :

        { 1: { 'name' : 'Raji', 'age' : 21 }, 2 : { 'name' : 'nandu' }, 3 :{'name' : 'Gopi'} }

Add Dictionary to Nested Dictionary :

        my_dict = { 1 : { 'name' : 'Raji', 'age' : 21 }, 2 : { 'name' : 'nandu' }}

        dict [3] = {'name' : 'Teju'}

        print(my_dict)

Output :

        { 1 : { 'name' : 'Raji', 'age' : 21 }, 2 : { 'name' : 'nandu' }, 3 : {'name' : 'Teju'} }

Delete Dictionary :

        my_dict = { 1 : { 'name' : 'Raji', 'age' : 21 }, 2 : { 'name' : 'nandu' }}

        del my_dict[2]

        print(my_dict)

Output :

        { 1 : { 'name' : 'Raji', 'age' : 21 }}

Key Points to Remember :

  • Nested Dictionary is an unordered collection of dictionary.
  • Slicing Nested Dictionary is not possible.
  • Like Dictionary , it also has key and value.
  • Dictionary are accessed using key.
