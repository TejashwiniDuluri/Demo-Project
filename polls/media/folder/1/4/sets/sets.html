Sets

        

A set is an unordered collection of items. Every element in a set must be unique and immutable. However, the set is mutable

Creating a Set

A set is created by enclosing the elements within the curly braces{}, separated by comma(,)

        my_set = {1,2,3}

        print(my_set)

Output:

        {1,2,3}

empty set is created using set() function without any arguments

        a = set()

        print(a)

Output:

        set([])

A set can have any no.of items and those may be of different types(int, float,tuple,string) but cannot have mutable elements like list, set or dictionary as its elements

        my_set = {1.0, "idiot", (3,6)}

        print(my_set)

Output:

        {1.0, 'idiot', (3,6)}

Set doesn't support duplicates

        my_set = {1,2,3,2,2,3,4,5,6}

        print(my_set)

Output:

        {1,2,3,4,5,6}

Updating a Set

add() method adds the element by sorting the set

        my_set = {1,3,7,8}

        my_set.add(5)

        print(my_set)

Output:

        {1,3,5,7,8}

update() method is used to add multiple elements by sorting. It takes list, strings, tuples and other sets as its arguments where duplicates are avoided in all cases

        my_set = {1,3,7,8}

        my_set.update([2,4,5,6])

        print(my_set)

Output:

        {1,2,3,4,5,6,7,8}

Deleting a Set

An item can be deleted from the set using remove() and discard() methods

Example:

        my_set = {1,3,7,8}

        my_set.discard(7)

        print(my_set)

        my_set.remove(8)

        print(my_set)

Output:

        {1,3,8}

        {1,3,7}

All items from the set can be removed using clear()

Example:

        my_set = {"idiot"}

        my_set.clear()

        print(my_set)

Output:

        set()

Set Operations

Union

returns the union of two sets and is performed using "|" operator

Example:

        A = {1,2,3}

        B = {4,5,6}

        print(A | B)

Output:

        {1,2,3,4,5,6}

Intersection

returns the set of elements that are common in both the sets and is performed using "&" operator

Example:

        A = {1,2,3}

        B = {2,3,5,6}

        print(A & B)

Output:

        {2,3}

Difference

returns the set of elements that are only in the first set and is performed using "-" operator

Example:

        A = {1,2,3,4}

        B = {3,4,5,6}

        print(A - B)

        print(B - A)

Output:

        {1,2,3}

        {5}

Symmetric Difference

returns the set of elements that are present in both the sets except those that are common in both and is performed using "^" operator

Example:

        A = {1,2,3,4}

        B = {3,4,5,6}

        print(A ^ B)

Output:

        {1,2,5,6}

Set Methods

add

adds a given element to a set, if the element is not present

Example:

        my_set = {1,2,3,4}

        my_set.add(5)

        print(my_set)

Output:

        {1,2,3,4,5}

        my_set = {1,2,3,4} ---------> doesn't add existing element

        my_set.add(2)

        print(my_set)

Output:

        {1,2,3,4}

copy

returns a copy of the set

Example:

        my_set = {1,2,3,4}

        your_set = my_set

        print(your_set)

Output:

        {1,2,3,4}

clear

returns an empty set

Example:

        my_set = {1,2,3,4}

        my_set.clear()

        print(my_set)

Output:

        set()

Set Functions

bin

converts and returns the binary equivalent string of a given integer

        t = bin(15)

        print(t)

Output:

        0b111

If the parameter is not an integer, it has to implement __index__() method to return an integer

        class flowers:

                tulips = 1

                lilly = 2

                rose = 3

                def __index__(self):

                        return self.tulips + self.lilly + self.rose

                        print(bin(flowers()))

Output:

        0b110

len

converts and returns the binary equivalent string of a given integer

Example:

        t = [1,2,3,4]

        print(len(t))

Output:

        4

Frozenset

Frozenset is the one which has the characteristics of a set, but its elements cannot be changed once assigned

Example:

        fruits = frozenset([1,2,3,4])

        print(fruits)

Output:

        [1,2,3,4]
