Conditional Statements

In programming decision making is made when specified condition is satisfied.Hence statements are executed only if that condition is satisfied.

Types of Conditional statements

  • if
  • if-else
  • elif
  • nested-if
  • switch

if statement:

  • if statement contains a logical expression using which data is compared.
  • logical expression is ended with colon.
  • all the statements inside if are indented.

Syntax:

  if  <expression> :

          statement(s)

Examples:

In the following example initially two variables are declared of type int and are compared using relational operator which is a logical expression. Upon satisfying that condition the statements inside if are executed.

if with int

Example:

        a=10
        b=20
        if b>a:
   print "miss b is greater than a"

Output:

miss b is greater than a

if-else statement:

  • if-else statement contains a logical expression using which data is compared and when expression evaluates to true,statements inside if are executed otherwise statements inside else part are executed.
  • logical expression after if and else statement are ended with a colon.
  • all the statements inside if and else are indented.

Syntax:

        if expression:

                statement(s)

        else:

                statement(s)

Examples:

Example of if-else with integers

        a=10
        b=20
        if b>a:

                print "miss b is greater than a"

        else:

                print "miss a is greater than b"

Output:

miss b is greater than a

elif statements:

  • Similar to the else, the elif statement is optional
  • In elif, there can be an arbitrary number of elif statements following an if
  • The elif statement allows you to check multiple expressions for TRUE
  • execute a block of code as soon as one of the conditions evaluates to TRUE
Syntax:

        if expression1:

                statement(s)

        elif expression2:

                statement(s)

        elif expression3:

                statement(s)

Example:

        a=100

        if a < 50:

                print(" idiot")

        elif a == 100:

                print(" donkey")

        else:

                print("stupid")

Output:

donkey

nested-if statements:

In a nested if construct, you can have an if...elif...else statement inside another if...elif...else statement

Syntax:

        if expression1:

                statement(s)

        if expression2:

                statement(s)

        elif expression3:

                statement(s)

        else:

                statement(s)

        elif expression4:

                statement(s)

        else expression1:

                statement(s)

Examples :

Example having an if...elif...else statement inside another if...elif...else statement

        a=100

        if a < 200:

                print(" idiot")

        if a == 150:

                print(" donkey")

        elif a == 100:

                print("great")

        else:

                print("mental, it's wrong!")

        else:

                print "Well done!"

Output:

idiot

great

Well done!

        a=[1,2,3,4]

        for x in xy:

                        print(x)

Switch Statement:

  • A switch statement is a multiway branch statement that compares the value of a variable to the values specified in case statements
  • Python language doesn't have a switch statement. It uses dictionary mapping to implement switch statement

Example:

        def Switch(argument):

                Switcher = {

                        0 : "zero",

                        1 : "one",

                         2 : "two",

                            }

                return Switcher.get(argument, "nothing")
