Functions

Functions in python are used to utilize the code in more than one place in a program, sometimes also called method or procedures. It is a block of code which only runs when it is called. You can pass data known as parameter, into a function. It can return data as result.

Creating a Function

We can define functions to provide the required functionality.

A function block begins with "def" keyword by the function name and parantheses(). We can define parameters inside these parantheses. The code block within every function starts with a colon(:) and is indented.

Syntax

        def functionname(parameters):

                "function_docstring"

                function_suite

                return [expression]

Example

        def my_function():

                print "fruit"

                return;

Calling a Function

Once a function is defined, it can be executed by calling that function

Example

        def my_function():

                print "fruit"

                return;

        my_function()

Output:

        fruit

Return function

                 return [expression_list]

                    c=a+b

Variable-length parameters

An asterick(*) is placed before the variable name that holds the value of all non-keyword variable arguments

Syntax

        def functionname(formal_args, *var_args_tuple):

                "function-docstring"

                function_suite

                return [expression]

Example

        def my_function(arg1, *vartup):

                print arg1

                for var in vartup:

                        print var

                        return;

        my_function("raji")

        my_function("nandu","teju")

Output:

        raji

        nandu

        teju

Lambda functions

The "lambda keyword" is used to create "anonymous functions"

Syntax

lambda[arg1[,arg2,...argn]] : expression

Examples

Here is the example of an anonymous function that returns the double value of "i"

        my_fun= lambda i : i + 3

                print my_fun(2)

Output:        

        4

Lambda defined function can have more than one input

Example

        my_fun= lambda i, j : i + j

                print my_fun(2, 1)

Output:

        3

Scope of Variables

Accessing a variable from a location in a program, depends on the location where the variable is declared

The following are the two basic scopes of variables:

  • Global Variables
  • Local Variables

Global Variables

A variable which is declared outside of the function is known as global variable, which means that a global variable can be accessed inside or outside of the function

Example

        Y = "plants"

        def trees():

                print "Y inside :", Y

        trees()

        print "Y outside :", Y

Output:

        Y inside : plants

        Y outside : plants

Local Variables

A variable which is declared inside the function body is known as Local Variable

Example

        def trees():

                Y = "plants"

                return Y

        trees()

Output:

        plants

Pass by reference & value:

The most widely used and easy to understand approaches to parameter passing among languages are   pass-by-reference and pass-by-value.Python is “pass-by-object-reference”.

Pass-by-reference :

In pass-by-reference ,the variable is passed directly into the function, and its contents implicitly come with it. Inside the function context, the argument is essentially a complete alias for the variable passed in by the caller. They are both the exact same variable, and therefore also refer to the exact same object in memory.

Example :

        def  den(mylist):

                    mylist=[1,2,3,4];

                    print(“values inside the function:”, mylist)

                    return

        mylist=[‘a’, ‘b’, ‘c’]

        den(mylist):

                print(“values outside the function:”, mylist)

Output:

values inside the function: [1,2,3,4]

values outside the function: [‘a’, ‘b’, ‘c’]

Function Arguments

*args

*args in python is used to pass a variable number of arguments to a function. It is used to pass a non-keyworded, variable-length argument list.

  •  The variable that we associate with the * becomes an iterable meaning you can do things like iterate over it, run some higher order functions such as map and filter, etc.

Example

        def adder(*num)

                    sum=0

                    for i in num:

                                sum+=i

                    return sum

        adder(1,2)

        adder(2,3,4)

output :

        3

        9

**kwargs :

**kwargs in python is used to pass a keyword, variable length argument list.we use kwargs with double star before parameter to define type of argument.

The arguments are passed as a dictionary and these arguments make a dictionary inside function with name same as the parameter excluding double asterisk **. 

Example

def func(**kwargs):

     if kwargs is not None:

          for key,value in kwargs.iteritems():

          print key,value

          return key,value

func(name=”raji”)
