Files

File is a location in harddisk that permanently stores the data. In programming, files plays a vital role as they store data in them even when computer is turned off Whereas data stored in RAM is erased as it is a volatile storage. Python treats different files in different ways, for example, text files contain only text  and binary files are those which are understood by computer.

Example:

teja.txt(text file)

        hi teja

        how are you

        had ur dinner?

File Operations

In python file operations are as follows:

1) open a file

2)read/write/append/other operations

3)close

open a file:

To perform read ,write or any other operations on files, first it needs to be opened. open() function is used to open files which takes two arguments, filename/filepath and mode in which file is to be opened.

Modes:

r

Read file only you cannot edit or write into it.

w

Write  into file if exists or truncate file if doesn’t exists.

rb

Read file only in binary mode.

a

Append at end of file if exists or else creates a new file and append.

r+

Both read and write into a file.

wb

Write file only in binary mode.

ab

Opens a file in binary mode for appending.

wb+

Both read and write in binary mode.if file exista it allows for read and write or else creates a new file.

        By default every file is opened in read mode if no other mode is specified.

Example:

        my_file=open(“teja.txt”)    #opening by using file name.

        my_file=open(“c:/downloads/teja.txt”)         #opening using file path.

        my_file=open(“teja.txt”,’r’)     #opening file in read mode.

        For i in my_file:

                print i        #it prints each line one by one from my_file.

File can also be opened in text or binary form. By default it is opened in text mode which returns strings when reading from it and returns bytes when opened in binary mode that comes into pictures in case of image or executive files.

So, when dealing with files in text mode it is suggested to provide its encoding type which is’UTF-8’ for linux and ‘cp1252’ for windows.

        my_file=open(“teja.text”,mode=’r’,encoding=’UTF-8’)

close file:

Every file that is opened need to be closed. close() method is used to close any file. Whenever close() method is encountered, all the resources associated with the file are made free.

        my_file=open(“teja.txt”,’w’)

        #file operations to be performed

        my_file.close()

        

read a file:

read() method is used to read a file .

   1)  my_file=open(‘teja.txt’,’w’)

        my_file.read()

output:”hi teja\nhow are you\nhad ur dinner?\n”

In the above example shown, first file is opened in read mode and then it is read. Finally, it returns strings  separated by newline until it reaches end of the file.

        

        my_file.read()

output:’ ’

further, if file is read then it returns an empty string as it is completely read.

        

read(n):   This method reads n no.of characters from the given file.

readline(n):   This method returns n no.of lines from the given file.

Readlines():    This method reads all the lines of file.

        

   2)  my_file=open(‘teja.txt’,’w’)

         my_file.read(2)

output:”hi”

as the size is mentioned, only that specific number of data is returned.

        

         my_file.readline(2)

output:”hi teja\nhow are you\n”

as shown above, two lines of data is been read, separated by new line character i.e \n.

        

        my_file.readlines()

output:’hi teja\n’,’how are you\n’,’had ur dinner?\n’

above method reads all the lines of file at a time.

        

        my_file.read()

output:’’

further reading file returns an empty string as shown above.

Write to a file:

         write ()  method is used to write into a file. While opening the file, mode is specified as write(‘w’). The issue with the write is that it overwrites into the file if it already exists or else creates a new file.

   1)  my_file=open(‘teja.txt’,’w’)

        my_file.write(‘what special u made today?\n’)

        my_file.close()

        my_file=open(‘teja.txt’)

        text=my_file.read()

        print text

output:   What special u made today?

As shown above, all the statements in the file teja.txt are replaced by above statement written into it.

         

   2)  my_file=open(‘teja.txt’,’w’)

        text=[“namaste”,”namaskar”]

        my_file.writelines(text)

        my_file.close()

writelines() method is used to write multiple lines of text into the file as shown above.

append to a file:

        The difference  between write and append is that, append attaches the text to the end of the file content whereas write overwrites the existing data.

        my_file=open(‘teja.txt’,’a’)

        my_file.write(“nice to meet you”)

        my_file.close()

output:    hi teja

                how are you

                had ur dinner?

                nice to meet you

with statement:

        python allows multiple open statements using with statement, which creates a context manager that closes the file automatically when the work is done with it.

        1) With open(‘teja.txt’) as file:

                for line in file:

                        print line

        2) With open(‘teja.txt’,’w’) as file:

                file.write(“hlo teja”)
