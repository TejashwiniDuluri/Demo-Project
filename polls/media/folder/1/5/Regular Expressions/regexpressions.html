Regular Expressions

Regular Expressions  are a set of strings that are used to find and replace patterns that are matching with the strings. Regular Expressions are also known as regex or regexp.

In python, we have a module called “re” which supports regular expressions. So, inorder to use regular expressions we need to import “re” library.

The following is used to import the module “re”

        

        Import re

Methods of Regular Expressions

re.match(pattern, string)

  • finds for the match at the beginning of the string
  • returns the “matching object” if the string matches the pattern of the regular expression and returns “None” if the string doesn’t match the pattern
  • in the case of multi-line mode, the re.match() method will consider the beginning of the string but not the beginning of the each line.
  • usage of ‘r’ at the beginning of the pattern string indicates python raw string

 Example:

pattern = r"Chocos"

string = "Chocos"

if re.match(pattern, string):

    print("Match!")

else:

    print("Not a match!")

Output:
  Match!

re.search(pattern, string)

  • finds for the match anywhere in the string
  • takes an expression containing pattern and string and searches if pattern matches the string
  • returns the “matching object” if match is found and returns “None” if match is not found
Example:

patterns = [‘computer’, ‘games’]

string = “engineer”

for pattern in patterns:

        if re.search(pattern, string):

                print ‘match’

        else:

                print ‘no match’

Output:
 no match

The following are the methods which both match and search methods performs:

group()

To print the matching string, “group” is used

Example:

average = re.match(r’fan’, ‘fan is fixed to the ceiling’)

print average.group()

Output:
 fan

span()

returns a tuple with the start and end positions of the string

Examples:

average = re.search(’fan’, ‘fan is fixed to the ceiling’)

print average.span()

Output 
(0, 3)

average = re.search(’fan’, ‘fan is fixed to the ceiling’)    ----> #to know the start position of the string

print average.span()[0]

Output
  0

average = re.search(’fan’, ‘fan is fixed to the ceiling’)  -----> #to know the position of the string

print average.span()[1]

Output  3

start()

To know the beginning position of the matching string, the start() is used

Example:

average = re.match(r’fan’, ‘fan is fixed to the ceiling’)

print average.start()

Output:
  0

end()

To know the end position of the matching string, the end() is used

Example:

average = re.match(r’fan’, ‘fan is fixed to the ceiling’)

print average.end()

Output:

  3

re.findall(pattern, string):

returns a list containing the no.of occurences of all the matching patterns of the string

Example:

average = re.findall(r’fan’, ‘fan is fixed to the ceiling which works as a fan’)

print average.end()

Output:

  [‘fan’, ‘fan’]

re.split(pattern, string):

splits the given string according to the occurence of pattern

Example:

average = re.split(‘x’, ‘fan is fixed to the ceiling fan’)

print average

Output

  [‘fan is fi’, ‘ed to the ceiling fan’]

average = re.split(‘i’, ‘fan is fixed to the ceiling fan’)  ----> performs split wherever there is “i”

print average

Output

  [‘fan’ ‘s f’, ‘xed to the ce’, ‘l’,’ng fan’]

average = re.split(‘i’, ‘fan is fixed to the ceiling fan’, maxsplit=2)  ----> performs split for 2 times where there is “i”

print average

Output

  [‘fan’ ‘s f’, ‘xed to the ceiling fan’]

re.sub(pattern, repl, string):

  • Searches for the pattern and replaces the pattern with a new sub string.
  • If the pattern is not found, it remains unchanged

Example:

average = re.sub(‘icecream’, ’chocolates’, ‘i love icecream’)

print average

Output

  ‘i love chocolates’

re.subn(pattern, repl, string):

  • returns a tuple containing the new string and the replacements that were performed
  • If the pattern is not found, it remains unchanged

Example:

average = re.sub(‘icecream’, ’chocolates’, ‘i love icecream’)

print average

Output

  (‘i love chocolates’, 1)

re.fullmatch(pattern, string):

  • Searches for the match between the pattern and the string
  • If the pattern is not found, it remains unchanged

Example:

average = re.sub(‘icecream’, ’chocolates’, ‘i love icecream’)

print average

Output:

  ‘i love chocolates’

re.finditer(pattern, string):

  • returns an iterator that is matching with the matching object instances for the regular expression pattern in the string
  • empty matches are also included and the string is scanned from left-right and the matches are returned in the order found

Example:

average = re.sub(‘icecream’, ‘is icecream lovely’)

print average

Output

  <callable-iterator object at 0*7f3dd02d3910>

re.purge():

clears the regular expressioin cache

re.escape():

generates a backslash before every non alpha numeric character other than alpha-numeric characters

Example:

 average = re.escape(‘bottle is filled with water’)

print average

Output

bottle\ is\ filled\ with\ water

re.error():

raises an exception if an error occurs

Example:

average = re.error(‘icecream’, ‘i love chocos’)

print average/0

o/p: TypeError: unsupported operand type(s) for /: 'error' and 'int'

re.compile():

we can compile pattern into pattern objects that have methods for searching patterns and for string substitutions

Example:

import re

sweet= re.compile(r"[^A-Za-zs.]")

name = raw_input ("enter name: ")

while sweet.search(name):

print "enter correct name!"

name = raw_input ("enter name: ")

Output

enter name: sun

Compilation Flags

                       Flag

                              Description

                      ASCII, A

Makes several escapes like \w, \b, \s and \d match only on ASCII characters with the respective property

                    DOTALL, S

Makes . match any character, including newlines

                 IGNORECASE, I

Does case-insensitive matches

                    LOCALE, L

Does a locale-aware match

                 MULTILINE, M

Multi-line matching, affecting ^ and $

        VERBOSE, X(for ‘extended’)

Enable verbose RE’s, which can be organized more cleanly and understandably

Characters

The following are the general characters which are used to generate an expression to represent the required string

  .

displays any single character except newline ‘\n’

Example:

import re

average = re.findall(r’.’, ‘fan is fixed’)

print average

Output

[‘f’, ‘a’, ‘n’, ‘ ’, ‘i’, ‘s’, ‘ ’, ‘f’, ‘i’, ‘x’, ‘e’, ‘d’]

In the above example, space is also extracted

\w

displays each character by excluding the space

Example:

import re

average = re.findall(r’\w’, ‘fan is fixed’)

print average

Output

[‘f’, ‘a’, ‘n’, ‘i’, ‘s’, ‘f’, ‘i’, ‘x’, ‘e’, ‘d’]

\W

displays non alpha numeric characters

Example:

import re

average = re.findall(r’\W’, ‘fan is fixed to @#’)

print average

Output

[‘ ’, ‘ ’, ‘ ’, ‘ ’, ‘@’, ‘#’]

*

displays 0 or more occurrences of a pattern

Example:

import re

average = re.findall(r’\w*’, ‘fan is fixed’)

print average

Output

[‘fan’, ‘ ’, ‘is’, ‘ ’, ‘fixed’, ‘ ’]

+

displays 1 or more occurrences of a pattern

Example:

import re

average = re.findall(r’\w+’, ‘fan is fixed’)

print average

Output

[‘fan’, ‘is’, ‘fixed’]

?

displays 0 or 1 occurrences of a pattern

Example:

import re

average = re.findall(r’\w+’, ‘fan is fixed’)

print average

Output

[‘fan’, ‘is’, ‘fixed’]

^

returns the word from the start of the string

Example:

import re

average = re.findall(r’^\w+’, ‘fan is fixed’)

print average

Output

[‘fan’]

$

returns the word from the ending of the string

Example:

import re

average = re.findall(r’\w+$’, ‘fan is fixed’)

print average

Output

[‘fixed’]

\s

displays white space characters

Example:

import re

average = re.findall(r’\s’, ‘fan is fixed’)

print average

Output

[‘ ’, ‘ ’]

\S

displays non-white space characters

Example:

import re

average = re.findall(r’\S’, ‘fan123 is fixed’)

print average

Output

[‘f’, ‘a’, ‘n’, ‘1’, ‘2’, ‘3’, ‘i’, ‘s’, ‘f’, ‘i’, ‘x’, ‘e’, ‘d’]

\d

displays digits

Example:

import re

average = re.findall(r’\d’, ‘fan123 is fixed’)

print average

Output

[‘1’, ‘2’, ’ 3’]

\D

displays non-digits

Example:

import re

average = re.findall(r’\D’, ‘fan123 is fixed’)

print average

Output

[‘f’, ‘a’, ‘n’, ‘ ’, ‘i’, ‘s’, ‘ ’, ‘f’, ‘i’, ‘x’, ‘e’, ‘d’]

( )

returns matched text by grouping the regular expressions

Example:

import re

average = re.findall(r’\d{2}-\d{2}-(\d{4})’, ‘fan 86-905 24-04-2018, power 93-297 12-08-2016’)

print average

Output

[‘2018’, ‘2016’]

[..]

displays the character that is matching with the string in the square bracket[]

Example:

import re

average = re.findall(r’[f]\w+’, ‘fan is fixed’)

print average

Output

[‘fan’, ‘fixed’]

\b

displays exactly two consecutive characters from the start of the string

Example:

import re

average = re.findall(r’\b\w.’, ‘fan is fixed’)

print average

Output

[‘fa’, ‘is’, ‘fi’]

@

displays the word that lies immediately next to ‘@’

Example:

import re

average = re.findall(r’@\w+’, ‘fan@kalam i@yummy love@relations’)

print average

Output

[‘@kalam’, ‘@yummy’, ‘@relations’]
