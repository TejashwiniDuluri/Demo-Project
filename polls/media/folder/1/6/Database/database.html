Database

Database is a collection of information that can be used, managed, updated and deleted and are classified according to the organizational approach. Python’s standard database’s  interface is python DB – API, which provides a database application programming interface

DB – API

The DB – API allows to write python scripts to access any database engine. It is independent of any database engine and its implementation for MySQL  is MySQLdb. It includes the following:

  • Importing the API module.
  • Acquiring a connection with the database
  • Issuing SQL statements and stored procedures.
  • Closing the connection

MySQL Database

To connect database server from python, MySQL database is used as an interface. It implements the python database API v2.0

Implementation

Inorder to use it with python, you first have to install it and then use the following code to implement MySQL within your program

        import MySQLdb

If MySQLdb is not installed, it raises an error. To install it again, use the following command

For Python:

        pip install MySQL-python

For Ubuntu:

        $ sudo apt-get install python-pip

Creating MySQL Database

Firstly, login to the server using the following command

        mysql -u root

On success, it returns a connection id and the server version, else it returns an error

Now, create a database using the following command

        create database db_name;

Example:

        create database flowers;

Inorder to know about the available databases, use the following command

        show databases;

Now the database is created, to use it, run the following command

        USE db_name;

Example:

        USE flowers;

it returns “database changed”

Now, create table for the database using the following command

        create table table_name;

Example:

        create table types(id INT, name VARCHAR(20));

The following comand is used to know if the table is created

        SHOW TABLES;  

Give values to the fields in the table using the following command

        INSERT INTO TABLE_NAME()VALUES();

Example:

        INSERT INTO types(id, name)VALUES(1, “viv”);

if you want to insert multiple values,

        INSERT INTO types(id, name)VALUES(2, “nan”);

Use the following command to display the table

        SELECT *FROM table_name;

Example:

        SELECT *FROM types;

                           id

                          name

                           1

                           viv

                           2

                           nan

To create user, use the following command

        create user user_name;

Example:

        create user mouse;  

To grant all previleges while assigning the password, use the following command

        grant all on db_name.* to ‘db_user’@’localhost’ identified by ‘db_password’;

Example:

        grant all on flowers.* to ‘mouse’@’localhost’ identified by ‘1234’;

The above example grants all the privileges but if you want to limit them, use the following command

        grant select, insert, update, delete on db_name.* to ‘db_user’@’localhost’ identified by ‘db_password’;

Example:

        grant select, insert, update, delete on flowers.* to ‘mouse’@’localhost’ identified by ‘1234’;

Database Operations

Insert

creates a record into a table

Syntax:

        INSERT INTO table_name()VALUES();

Example:

        INSERT INTO types(id, name)VALUES(1, “viv”);

Update

updates a record in a table

Syntax:

        UPDATE table table_name SET name = %s WHERE name = %s;

Example:

        UPDATE types SET name = 'ted' WHERE name = “viv”;

        SELECT *FROM types;

                          name

                           2

Delete

deletes the records from the table

Syntax:

        delete from table_name WHERE name= %s;

Example:

        delete from types WHERE name= %s;

Creating a Connection Object

To connect to the database, connect() method is used

Syntax:

        db = MySQLdb.connect(“localhost”, “user”, “password”, “database”)

connect() method returns a connection object on success else it returns an operational error

Example:

        import MySQLdb as my

        db = MySQLdb.connect(localhost =  “host”,

                                     user =“root”,

                                     password = “  ”,

                                     database = “db”)

        db

output:    <_mysql.connection open to ‘localhost’ at 5620daaf88a0>

Creating a Cursor Object

Before interacting with database server, a cursor object must be created. Cursor is used to submit various SQL statements to a database server

Syntax:

        connection_object.cursor()

It returns cursor object on success, else returns an error

Example:

        import MySQLdb as my

        db = MySQLdb.connect(“localhost”: “host”,

                                     “user”: “user”,

                                     “password”: “  ”,

                                     “database”: “db”)

        cursor = db.cursor()

        cursor

output:  <MySQLdb.cursors.Cursor at 0x7f6b944c5650>

Interacting with database

To execute sql queries cursor object makes use of execute() method

Syntax:

        cursor.execute(sql);

It returns no.of effected rows on success, else returns an error

Example:

        import MySQLdb

        db = MySQLdb.connect(“localhost”: “host”,

                                     “user”: “user”,

                                     “password”: “  ”,

                                     “database”: “db”)

        cursor = db.cursor()

        cursor

        no_of_rows = cursor.execute(“select *from table_name”);

        no_of_rows

output:  displays the no_of_rows in the table

Closing the Connection

 To disconnect the database connection, close() method is used

Syntax:

        connection_object.close()

Example:

        import MySQLdb         

        db = MySQLdb.connect(“localhost”: “host”,

                                     “user”: “user”,

                                     “password”: “  ”,

                                     “database”: “db”)

        db

        db.close()

output:   closes the database db
