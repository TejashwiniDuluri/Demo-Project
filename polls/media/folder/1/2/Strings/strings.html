String Slicing:

Slicing is used to get of letters from string [x:x]is used when slicing a string.

var = hi john'

var[0]-------->one character of given string 'h'

var[0:1]------->one character of string form beginning (0) 'h'

var[0:4]------->first 4 character from given string 'hi j'

var[:3]-------->first three characters 'hi'

var[3:]--------> all characters except first three 'john'

var[:-4]-------->all except last three 'hi j'

Note: Accessing a character out of range will raise index error & index must be integer not float (or) others or else raises type error.

Change (or) Delete String:

Strings are immutable. This means that elements of a string cannot be changed once it has been assigned. We can simply reassign different strings to the same name.

var = "teju"

var[3] = 'u'

print var

Output : Type Error: 'str' object does not support item assignment

var = "teju"

print var

Output :"teju"

We cannot delete or remove characters from a string. But deleting the string entirely is possible using the keyword del.

var = "teju"

del var[1]

print var

Output :Type Error: 'str' object doesn't support item deletion

Methods of String:

Python has a set of built-in methods that you can use on strings.

replace()

Returns a string where a specified value is replaced with a specified value

islower()

Returns True if all characters in the string are lower case

isupper()

Returns True if all characters in the string are upper case

capitalize()

Converts the first character to upper case

startswith()

Returns true if the string starts with the specified value

endswith()

Returns true if the string ends with the specified value

len()

Returns the length of the string
