Multiplication

Multiplies the two operands

        print(“c is:”, c)

Output:

Division

Divides the left operand by the right operand

        print(“c is:”, c)

        print(“c is:”, c)

Comparison Operators

Comparison operators compares values which returns True or False according to the condition.

Examples

Greater than

returns True if left operand is greater than the right operand

        a = 5

        b = 3

        c = a > b

        print(“c is:”, c)

Output:

        True

Less than

returns True if left operand is less than the right operand

        a = 5

        b = 3

        c = a < b

        print(“c is:”, c)

Output:

        False

Equal to

 returns True if both the operands are equal

        a = 5

        b = 3

        c = (a == b)

        print(“c is:”, c)

Output:

        False

Not equal to

returns True if both operands are not equal

        a = 5

        b = 3

        c = (a != b)

        print(“c is:”, c)

Output:

        True

Greater than or equal to

returns True if left operand is greater than or equal to the right operand

        a = 5

        b = 3

        c = a >= b

        print(“c is:”, c)

Output:

        True

Less than or equal to

returns True if left operand is less than or equal to the right operand

        a = 5

        b = 3

        c = a <= b

        print(“c is:”, c)

Output:

        False

Bitwise Operators

Bitwise operators operates bit by bit which work on operands as string of binary digits.

For example, 2 is 10 in binary and 6 is 110.

To perform bitwise operations, Consider x = 10 (0000 1010 in binary) and y = 4 (0000 0100 in binary)

Examples

Bitwise AND

        

performed using “ & ”

        x&y=0(0000 0000)

Bitwise OR

                

performed using “ | ”

        

         x|y=14(0000 1110)

Bitwise NOT

        

performed using “ ~ ”

        

        ~x=-11(1111 0101)

Bitwise XOR

performed using “ ^ ”

        

        x^y=14(0000 1110)

Bitwise right shift

                

performed using “ >> ”

        x>>2=2(0000 0010)

Bitwise left shift

                

performed using “ << ”

        

        x<<2=40(0010 1000)

Special Operators

 “Identity operator and the membership operator” are some special operators.

Identity Operators

        “is and is not” are the identity operators which are used to check if two values are located in the same memory. Two variables that are equal does not imply that they are identical.

is

        returns True if the operands are identical

Example

        a1 = 5

        b1 = 5

        a2 = ‘hi’

        b2 = ‘hi’

        print(a1 is b1)

Output:

        True

is not

        returns True if the operands are not identical

Example

        a1 = 5

        b1 = 5

        a2 = ‘hi’

        b2 = ‘hi’

        print(a1 is not b2)        

Output:

        True

Membership Operators

“in and not in” are the membership operators which are used to check for the presence of value or variable in a sequence (string, list, tuple, set and dictionary). But in dictionary, presence of key can only be checked, but not the value.

in

        returns True if value is found in the sequence

Example

        a = “hello”

        b = [1,2,3]

        print(1 not in a)

Output:

        True

not in

        returns True if value is not found in the sequence

Example

        a = “hello”

        b = [1,2,3]

        print(‘e’ in a)

Output:

        True

Identifiers

  • Identifiers can be a combination of letters, numbers that may be lowercase (a to z) or                          uppercase (A to Z) or digits (0 to 9) or an underscore (_)
  • An identifier cannot start with a digit
  • Keywords cannot be used as identifiers
  • Special symbols like !, @, #, $, % etc. cannot be used in identifier
  • Identifier can be of any length
