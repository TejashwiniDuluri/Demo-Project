Variables

What is a variable?

Variable is a reference to a value. A value can be any string,number,character,int,float,etc..

Syntax :

variable name = value

Example:

a = "a"

So value can be assigned to variable using "=" assignment operator.

Rules :

  • Variable name should not start with number.
  • It should only contain alphanumeric character.
  • Variable names are case sensitive [ Example = age,Age,AGE ]
  • Keywords cannot be used as variable names.

Multiple Assignments :

Python allows multiple assignments at a time with different types of values. Allows you to assign single value to several variables.

Example:

a = b = c = 1 --------> [But is of no use]

a , b , c = 1, 2, "John"

Accessing values to variables :

Variable value can be accessed using its name.

Example:

a = 5

print a

Output :5

Redeclaring Variable :

A variable can be redeclare using different values.

Example:

a = 10

print a

Output :10

a = "John"

print a

Output :John

Delete Variable :

To free memory space (or) To free allocated memory space.

Numbers :

It sub divided into

  • int
  • Float
  • long
  • complex

Int :

Value of integer is not restricted. Updating integer values

Example:

a = 10

a = a+1

Output :11

Note:In python 2 version int & long int is used ,where as in python 3 version only int is used. It can expand up to the available memory limit.

Float :

Float point value can be positive ,negative and can contain one (or) more decimal.

Creating Float Value :

a = 10.0

Updating Float Value :

a = 10.0 + 1.0

print a

Output :11.0

Delete Float Value :

delete a

Output :a will be deleted

Complex :

A complex number consists of an ordered pair of real floating-point numbers denoted by x+yj,where x and y are the real numbers and j is the imaginary unit.

Example:

a+4j
