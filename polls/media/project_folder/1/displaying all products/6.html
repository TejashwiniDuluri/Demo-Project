Displaying all products

  • Creating a view for displaying all products .

Functionality of this view is to get all the product objects and send context object to template .

Hint :

Retrieve all the products using following Query and send the list as context

Model.objects.all()

Code :

def ProductsList ( request ) :

prod_list = Products.objects.all()

context = {‘ prod_list ’ : prod_list}

return render (request , ‘orders / procucts.html’ , context )

  • writing an url for associated View (all products) .

On requesting this url it will send a request to the all products view , inturn view will send the required context to template .

Hint :

import the view from views .py and link it to url

Code :

from order . Views import ProductList

url (‘products’ , views.ProductList , name = ‘ products ’ )

  • Creating a template to display all products list(All products list).

This template will be linked to the above (all products )view and gets context objects as response from view and displays all the products details .

Hint :

Recieves context object productList from view as prod_list

code :
