Creating Product

  • creating view to Create product .

Functionality of this view is to load the form template in order to save the fom data and validate and to create a new product object .

Hint :

Import the modelform from from.py to load and save the form data.

Code:

from forms.py import ProductForm

def product_create (request ) :

form = ProductForm( request.POST )

if request.method == ‘POST’:

if form.is_valid ( ):

form.save()

else :

form = ProductForm()

context = { ‘form’ : form }

return render (request , orders/ProductCreate.html , context)

  • Create a Form using model

Hint :

import product model from models.py and create a form

code :

from order.models import Product

class ProductForm (ModelForm):

class Meta :

model = Product

fields = [‘Product_name’, ‘Product_details’ , ‘Price ’ , ‘Active’]

Create a form template to Create Product

Extend this form for index.html .

Create html template with below attributes or rather use django model form which takes care generating html body for submiting data .

Hint :

By using ModelForm , there no need to write html for every attribute .Simply place

{{form. as_p }} which will take care of generating html form in template .

--> Form contains

  • Product_name .

  • Product_details .

  • Price .

  • Active .

Note : if using model form the template structure will be the below .

<form method = POST >

{% csrf_token %}

{{ form.as_p }}

<button type="submit" class="btn btn-success">Submit</button>

</form>
