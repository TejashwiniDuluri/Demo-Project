Starting app

Below command is used to startapp

-- python manage.py startapp sampleapp

Appending app to Installed Apps

In settings.py configure installed apps .

INSTALLED_APPS = [

......

' django.contrib.staticfiles ' ,

' sampleapp ' ,

]

Steps To be added in App

  • Appending app(orders) to INSTALLED_APPS list in settings.py .

  • Create a urls.py file in app ( orders ) .
