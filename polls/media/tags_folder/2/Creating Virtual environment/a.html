1. Creating Virtual environment .

Python 2

-- virtualenv venv -p python

-- Here venv is the name of Virtual Environment

-- Created Virtual environment of pyhton 2.x

Activating Virtual environment

-- $ : Source venv/bin/activate

Deactivating Virtual environment

-- (venv) $ : Deactivate

Installing Django inside virtual environment

-- Pip install django == ' version number ' ( 1.9 or 1.10 or 1.11)

By default in python2.x vitualenv , django installation will be under version 2.0

Python3

-- virtualenv django -p python3

-- Here django is the name of Virtual Environment .

-- Created Virtual environment of pyhton 3.x

Activating Virtual environment

-- $ : Source venv/bin/activate ( ubuntu )

Deactivating Virtual environment

-- ( django ) $ : Deactivate

Installing Django inside virtual environment

-- Pip install django == ' version number ' ( 2.x)

By default in python 3.x vitualenv , django installation will be grater version of 2.0
